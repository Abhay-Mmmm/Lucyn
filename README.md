# Lucyn

> AI-powered engineering intelligence for modern software teams.

Lucyn helps engineering teams work smarter by providing insights into team health, velocity, and collaboration patterns.

---

## About

Lucyn is a SaaS platform designed to help engineering leaders understand their team's productivity and well-being through data-driven insights.

## Features

- **Team Analytics** - Monitor team health and performance metrics
- **Code Insights** - Track repository activity and contribution patterns  
- **Integrations** - Connect with GitHub, Discord, and other developer tools
- **Smart Dashboard** - Visualize engineering metrics in an intuitive interface

## Technology

Built with modern web technologies including Next.js, TypeScript, and PostgreSQL.
